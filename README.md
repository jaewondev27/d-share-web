# D-Share Web Host

LAN 전용 웹 호스트 — **별도 서버 없음**

## GitHub Pages (이 페이지)

https://jaewondev27.github.io/d-share-web/

1. 클라이언트 앱 실행
2. 앱에 표시된 **IP** 입력 → 웹 호스트 열기
3. **6자리 코드** 입력 → 화면 공유

## 직접 접속 (권장)

클라이언트 앱에 표시된 주소를 PC/태블릿 브라우저에서 열기:

`http://클라이언트IP:3849/`

같은 Wi-Fi · 코드만 입력 · 릴레이/클라우드 서버 불필요
